Reminder:
*Node.js and MongoDB should be installed on your device*
* type "node -v" to check if you installed Node.js, it should show the version number of Node.js


1. Open a terminal or command prompt
2. Navigate to the directory where 'server.js" file is located. You can use cd command to change directions.
3. Type "npm install" to install all the dependencies.
4. Type "node server.js" to start the server.
5. Open your web browser and visit http://localhost:3000.
6. You should see the application's login page.
